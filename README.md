# Aeroplane Tracker 

Программа для отслеживания самолётов в реальном времени над выбранными странами. Использует OpenSky Network API и Nominatim OpenStreetMap API.

## Функциональность

- Поиск самолётов над указанной страной
- Сохранение данных в JSON-файл
- Фильтрация по стране регистрации
- Фильтрация по диапазону высот
- Вывод топ-N самолётов по высоте
